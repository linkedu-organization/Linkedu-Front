# Linkedu-Front

Desenvolvido com **React + Vite + TypeScript**.

## Pré-requisitos

- **Node.js** ≥ 18  
- **npm**


## Instalação

Na pasta do projeto (frontend):
        npm install

## Rodar 
        npm run dev


## Lint (ESLint)
        npx eslint src

## Cypress
        npx cypress open


** Cypress está configurado sem supportFile
